GO module

在较早前的版本，GO 需要配置 go path 才能工作，新版本中推出了 go module，只要文件夹中含有 go.mod 文件就能工作，也不需要包含 src，bin 这样的目录（1.11 版本以上才能使用 Go Module，1.13 版本以下 Go Module 默认关闭）

定义数量级

整形

在 go 中常用 uint 和 int 两种整形，两种整形都有相应的 8，16，32 以及 64 位，不同在于 uint 最小值为 0，而 int 的范围包含负数，比如：

  • uint8:无符号 8 位整型（0~255）
  • int8：有符号 8 位整型（-128~127）

特殊整形

  • uint: 32 位系统上就是 uint32，64 位上就是 uint64
  • int: 和 uint 同理
  • uintptr: 无符号整型，用于存放一个指针

注意：32位的机器上只能最高使用32位的整型，无法使用64位
