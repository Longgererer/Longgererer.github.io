获取窗口宽高

  • 网页可见区域宽高（不包括垂直和横向滚动条） ➡ 可见 element content + padding，会随窗口显示大小而改变
  • 网页可见区域宽高（包括边框） ➡ element content + padding + border-width，包括滚动条和边线（不包括浏览器本身的滚动条），会随窗口显示大小而改变， 注意：offsetWidth 只会显示可见区域宽度，但是 offsetHeight 显示的是元素的全部高度，包括屏幕外的
  • 网页正文宽高 ➡ 全部 element content，包括屏幕外的正文内容和因为 overflow 而不显示的内容，不包括滚动条
  • 屏幕可工作区域宽高 ➡ 屏幕宽高，包括滚动条，不包括系统任务栏
  • 屏幕分辨率
