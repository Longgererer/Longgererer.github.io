学了那么久的 js，有必要了解一些实用的原生 API 了

window 属性

closed

此属性返回一个布尔值，用于判断指定窗口是否关闭，常和 window.open() 方法一起使用：

控制台将输出 window closed

如果直接在当前页面上执行 window.closed 返回值为 false，因为当前窗口是打开的

document

此属性是对 Document 对象的只读引用，平常使用都是直接省略 window

frames

该选项返回一个数组，数组包含当前窗口中所有的 iframe

html:

js:

window.frames[0] 将获取第一个 iframe，更改其 location 属性将重定向到一个新的页面

history

改属性为一个对象，包含下面常用属性和方法

  • length
    返回历史列表中的网址数

  • back()
    加载历史列表中的上一个 url（后退）

  • forward()
    加载历史列表中的下一个 url（前进）

  • go(number/url)
    加载历史列表中的一个具体页面

innerWidth&innerHeight

可以通过：

来获取窗口的文档显示区的宽高（不包括浏览器工具栏和滚动条）

这两个属性不是只读的，可以修改 innerWidth 和 innerHeight 来改变窗口大小

localStorage

该属性允许在浏览器中存储键值对数据，用于长久保存浏览器的数据，过期时间为永久，除非手动删除，否则数据将会一直保存在本地

localStorage 是只读的

在浏览器中支持 localStorage 大小一般为5M，在不同浏览器中限制大小会有所不同

localStorage 的值类型限定为 String 类型，所以存储对象类型的时候需要先进行转换

localStorage 有下面两种写入方法：

  • storage["a"]=1 或 storage.a=1
  • storage.setItem("a",3)

location

location 属性

假设有这样一个 url：https://localhost:4040/index.html?a=1#fragment

  • hash
    返回一个 url 的锚部分（从#开始）
    location.hash // fragment

  • host
    返回主机名和端口
    location.host // localhost:4040

  • hostname
    返回主机名
    location.hostname // localhost

  • href
    返回完整的 url
    location.href // https://localhost:4040/index.html?a=1#fragment

  • pathname
    返回 url 路径名
    location.pathname // /index.html

  • port
    返回端口号
    location.port // 4040

  • protocol
    返回 url 协议
    location.protocol // https:

  • search
    返回 url 查询部分
    location.search // ?a=1

location 方法

  • location.assign(URL)
    加载一个新的文档

  • location.reload()
    重新载入当前文档

  • location.replace(url)
    载入新文档来替换当前页面

opener

改属性返回创建此窗口的窗口的引用

newWindow.opener 指向的是当前窗口，也就是 newWindow 的原窗口

screen

这个属性是一个对象，包含有关客户端显示屏幕的信息

  • availHeight
    返回屏幕的高度（不包括 windows 底部的任务栏，和浏览器本身窗口大小无关）

  • availWidth
    返回屏幕宽度

  • height
    返回屏幕的总高度（包括任务栏）

  • width
    返回屏幕的总宽度

screenLeft&screenTop

返回窗口相对于屏幕的 x 和 y 坐标（左上角），值为 Number 类型

screenX&screenY

和 screenLeft，screenTop 是一样的作用，只是 IE 不兼容 screenX 和 screenTop 这两个属性

pageXOffset&pageYOffset

pageXOffset 和 pageYOffset 属性返回文档在窗口左上角水平和垂直方向滚动的像素

sessionStorage

改属性用于创建一个本地存储的键值对，但和 localStorage 不同的是，在会话窗口关闭的时候会删除这些数据

和 localStorage 一样，大部分浏览器将存储上线设置为5M，存储的值也必须为字符串

window 方法

confirm()

该方法显示一个带有取消和确认按钮的对话框

getComputedStyle()

该方法返回一个对象，用于获取一个元素的 css 样式

接受两个参数，一个是元素，一个是伪类，如果没有伪类可以忽略或者传递 null

上述代码可以获取元素的背景颜色

open()

打开一个新的浏览器窗口或者查找一个已命名的窗口

接受四个参数：

  • url
  • 可选参数，指定 target 或者窗口名称，默认为 _blank 加载到一个新的窗口
  • 可选参数，一个逗号分隔列表，支持很多值，就不写出来了
  • 可选参数，为布尔值，如果为 true 则替换浏览历史中的当前条目，false 则在浏览历史中创建新的条目
